# Harper for Windows Command Prompt

A color scheme inspired by the art of [Charley Harper](http://en.wikipedia.org/wiki/Charley_Harper) (1922-2007)

![Screenshot](https://raw.github.com/idleberg/Harper-Windows-Command-Prompt/master/images/screenshot.png)

## Installation

1. Download the `.reg`file
2. Double-click the file to apply changes
